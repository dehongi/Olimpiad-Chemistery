::: details مشاهده پاسخ
ابتدا **عدد اکسایش فسفر در (K[PF_6])** را به‌دست می‌آوریم.

### (K[PF_6])

یون کمپلکس: ([PF_6]^-)

اگر عدد اکسایش فسفر = (x) باشد:
[
x + 6(-1) = -1
]
[
x - 6 = -1 \Rightarrow x = +5
]

پس **عدد اکسایش فسفر = +5** است.

---

### بررسی گزینه‌ها

**الف) (\mathrm{NaH_2PO_3})**
در یون (\mathrm{H_2PO_3^-}):
[
x + 2(+1) + 3(-2) = -1
\Rightarrow x = +3
]
❌ برابر نیست

**ب) (\mathrm{P_4})**
عنصر آزاد → عدد اکسایش = **0**
❌ برابر نیست

**ج) (\mathrm{H_3PO_3})**
[
x + 3(+1) + 3(-2) = 0
\Rightarrow x = +3
]
❌ برابر نیست

**د) (\mathrm{H_3PO_4})**
[
x + 3(+1) + 4(-2) = 0
\Rightarrow x = +5
]
✅ برابر است

---

**گزینهٔ د) (\mathrm{H_3PO_4})**
