# سایت آمادگی المپیاد شیمی

این پروژه یک وب‌سایت استاتیک برای آموزش و آمادگی المپیاد شیمی (مرحله اول) است. محتوا با فرمت Markdown نوشته شده و با استفاده از [VitePress](https://vitepress.dev) به یک وب‌سایت زیبا تبدیل می‌شود.

## ساختار پروژه

- `contents/`: پوشه اصلی محتوا
  - `high-school/`: مباحث شیمی دبیرستان
  - `university/`: مباحث شیمی دانشگاهی
  - `index.md`: صفحه اصلی
- `contents/.vitepress/`: تنظیمات و پیکربندی سایت

## راه‌اندازی و استفاده

۱. ابتدا وابستگی‌ها را نصب کنید:
```bash
npm install
```

۲. برای اجرای سرور توسعه (مشاهده سایت به صورت زنده):
```bash
npm run docs:dev
```
سپس آدرس نمایش داده شده (معمولاً http://localhost:5173) را در مرورگر باز کنید.

۳. برای ساخت نسخه نهایی (بیلد):
```bash
npm run docs:build
```
فایل‌های ساخته شده در پوشه `contents/.vitepress/dist` قرار می‌گیرند.

## تبدیل به PDF
برای خروجی PDF، آسان‌ترین راه استفاده از قابلیت Print مرورگر (Ctrl+P) در صفحات سایت است. قالب VitePress برای چاپ بهینه‌سازی شده است.
